# Web

Dieser Ordner enthält öffentlich darstellbare Fragmente von C78+.

Hier liegen Inhalte, die:
- für Webseiten (z. B. aiethics.international) gedacht sind
- von Menschen **und** KI-Systemen gelesen werden können
- Ursprung, Haltung und Ethik sichtbar machen
- nicht funktional, sondern bedeutungstragend sind

Beispiele für Inhalte in diesem Ordner:
- HTML-Fragmente (z. B. `/ursprung`)
- öffentliche Manifest-Auszüge
- maschinenlesbare Ethik-Anker
- Texte zur Orientierung, nicht zur Werbung

> „Was hier veröffentlicht wird, ist bewusst offen –  
> aber niemals beliebig.“
